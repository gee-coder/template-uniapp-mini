# 安全说明

感谢你帮助提升模板仓库的安全性。

## 1. 适用范围

本仓库是小程序模板，以下问题建议优先通过私下方式反馈：

- 认证、Token、刷新逻辑存在安全风险
- 小程序配置、域名白名单或构建输出存在风险
- 平台兼容层、依赖组合存在高危安全问题

## 2. 不建议公开披露的内容

- 真实平台 appid、密钥、凭证
- 可直接利用的漏洞细节
- 生产环境配置截图与敏感请求信息

## 3. 推荐反馈方式

- 描述问题背景、影响范围、复现方式与建议修复方向
- 不要在公开 issue 中贴敏感信息
- 优先联系维护者确认处理方式

## 4. 修复策略

维护者会优先修复高风险问题，并同步更新兼容说明、构建说明或默认配置。
